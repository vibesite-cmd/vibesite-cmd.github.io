# Vibe Core

A minimal Three.js 3D experience showing how to structure a WebGL project for vibesite submissions.

## Structure
- `index.html` - Loads the CDN Three.js + CSS + main JS
- `css/style.css` - Dark elegant styling matching the site vibe
- `js/main.js` - All Three.js scene, camera, lights, particles, and mouse interaction

## How to run locally
1. Open `index.html` in a modern browser (it uses Three.js via CDN).
2. Drag to rotate the orb.
3. Use mouse wheel to zoom.

## Submitting Three.js projects
- Same rules as other projects: self-contained folder under `/projects/your-name/`
- You can use Three.js via CDN (easiest for GitHub Pages) or bundle a local copy.
- Keep file size reasonable.
- Update `projects.json` with your metadata.

This example uses r134 of Three.js for broad compatibility.
