# Vibe Tool — Project Generator

This is a self-contained tool for generating new vibe-coded static projects.

## Structure (example of what we want)
- `index.html` — the UI
- `css/style.css` — all styles
- `js/main.js` — generator logic

## How to use
1. Open `index.html` in a browser.
2. Fill out the form.
3. Generate.
4. Copy the output files into a new folder under `/projects/` when submitting to vibesite.

This tool itself is an example of a useful "tool" type submission (as opposed to pure games or art).

Submit improvements or new generators the same way!
