# Vibe Lure

A simple mouse-controlled avoider game demonstrating a clean folder structure for vibe-coded submissions.

## Structure
- `index.html` - Main page (links to css + js)
- `css/style.css` - Styles and colors
- `js/game.js` - All game logic (canvas, loop, collisions, particles)

## How to test locally
1. Open `index.html` directly in your browser.
2. Move your mouse to control the lure.
3. Collect green orbs, avoid red triangles.

## How to submit similar projects
Copy this entire folder into `/projects/your-project-name/` in the main repo, update `projects.json`, and open a PR.

Keep projects self-contained and fun.
