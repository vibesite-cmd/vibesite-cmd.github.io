Submit a Project

Two ways to add your creation to the archive. Both are welcome.

1. GitHub Pull Request (best)

  1. Fork the vibesite repository
  2. Create folder /projects/your-project-slug/
  3. Add your self-contained index.html + assets
  4. Add your project to projects.json
  5. Open a Pull Request with a short description
Screenshot of forking the repo would go here

2. Simple web upload

  1. Go to the repo on GitHub
  2. Click Add file → Upload files
  3. Upload your project folder into /projects/
  4. Describe what it is and why it has the right vibe
  5. Open a Pull Request or Issue
Screenshot of the GitHub upload button would go here
Projects should be self-contained static experiences (one HTML file is ideal). We'll review and merge within a day or two. Keep the strange, the quiet, the beautiful.
