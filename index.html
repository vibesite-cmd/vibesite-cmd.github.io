Submit a Project

Two easy ways to share your vibe-coded work with the community.

1. Via GitHub (recommended)

  1. Fork this repository
  2. Create a new folder inside /projects/your-project-name/
  3. Add your full self-contained project (ideally a single index.html + assets)
  4. Update projects.json with your project metadata
  5. Open a Pull Request with a short description

2. Simple upload (no git needed)

  1. Go to the repo on GitHub
  2. Click Add file → Upload files
  3. Upload your project folder into /projects/ (or open an Issue with the code pasted)
  4. Describe what it is and why it has the right vibe
Projects should be self-contained static web experiences (HTML + CSS + JS). No external servers or complex builds. We'll review and merge quickly.
